割引ファクター公式

単一期間割引ファクター

DF = 1 / (1 + r/n)^(n*t)
ここで: DF = 割引ファクター、r = 年率、n = 複利頻度、t = 時間期間

連続複利

DF = e^(-r*t)
ここで: e = オイラー数 (≈2.718)、r = 年率、t = 時間期間

現在価値計算

PV = FV × DF
ここで: PV = 現在価値、FV = 将来価値、DF = 割引ファクター

主要概念

割引ファクターとは何ですか?

割引ファクターは、将来のキャッシュフローにその現在価値を得るために乗じるファクターです。将来受け取る$1の今日の価値を表します。

応用

  • • DCF (割引キャッシュフロー) 分析
  • • 債券評価
  • • 投資評価
  • • 財務モデリング

複利効果

より頻繁な複利は割引ファクターを削減します (現在価値を増加させます)。利息がより頻繁に複利されるため、より高い実効率につながります。

解釈

割引ファクターが0.826であることは、割引率と時間期間を考えると、将来受け取る$1が今日$0.826の価値があることを意味します。
