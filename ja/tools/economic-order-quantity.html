EOQ公式と情報

EOQ公式

EOQ = √(2DS/H)
ここで:
D = 年間需要
S = 1回の発注コスト
H = 1単位あたりの年間保有コスト

関連公式

  • • 年間発注回数: N = D/EOQ
  • • 発注間隔: T = EOQ/D × 365日
  • • 総コスト: TC = (D/Q)S + (Q/2)H + D×C

主な仮定

  • • 需要は一定で既知
  • • リードタイムは一定で既知
  • • 数量割引なし
  • • 在庫切れは許可されない
  • • 発注コストと保有コストは一定

利点

  • • 総在庫コストを最小化
  • • 発注コストと保有コストのバランスを取る
  • • 最適な発注頻度を提供
  • • キャッシュフロー管理を改善
  • • ストレージ要件を削減
