アルトマンZ-スコア

アルトマンZ-スコアモデルを使用して企業の破産リスクを評価します。5つの財務比率を組み合わせて財務困難を予測します。

アルトマンZ-スコア計算機

このプロ実績のある財務困難予測モデルを使用して、アルトマンZ-スコアを計算し、破産リスクを予測し、企業の財務健全性を評価します。

安全ゾーン: Z > 2.99
グレーゾーン: 1.8 < Z < 2.99
困難ゾーン: Z < 1.8

財務データ入力

貸借対照表項目

損益計算書項目

Z-スコア結果

A: 運転資本/総資産
B: 利益剰余金/総資産
C: EBIT/総資産
D: 株式時価総額/総負債
E: 売上/総資産

解釈

破産確率:

アルトマンZ-スコア公式

Z = 1.2A + 1.4B + 3.3C + 0.6D + 1.0E
A: 運転資本 / 総資産
B: 利益剰余金 / 総資産
C: 利息税控除前利益 / 総資産
D: 株式時価総額 / 総負債
E: 売上 / 総資産

リスクゾーン

安全ゾーン (Z > 2.99)
破産リスクが低い。企業は財務的に健全です。
グレーゾーン (1.8 < Z < 2.99)
中程度のリスク。注意深い監視が必要です。
困難ゾーン (Z < 1.8)
破産リスクが高い。直ちに対応が必要です。
注: アルトマンZ-スコアはもともと製造企業向けに開発されました。サービス企業と非公開企業向けの修正版があります。

使用のヒント

データ要件

  • • 最新の年間財務諸表を使用してください
  • • すべての入力でデータの一貫性を確認してください
  • • 運転資本 = 流動資産 - 流動負債
  • • EBITは利息と税金を除きます

制限事項

  • • もともと製造企業向けに設計されました
  • • サービス企業にはうまく適用されない可能性があります
  • • 他の財務指標と一緒に使用する必要があります
  • • 市場条件は精度に影響を与える可能性があります
