Formule et Informations

Formule REO Obligation Escomptée

REO = [(Valeur Nominale - Prix d'Achat) / Prix d'Achat] × (365 / Jours jusqu'à l'Échéance)

REO Obligation avec Paiements Périodiques

REO = 2 × [(1 + r/n)^n - 1]
où r = taux annuel, n = fréquence de capitalisation

Concepts Clés

  • Rendement Équivalent Obligation : Mesure de rendement standardisée pour comparer les obligations
  • Obligation Escomptée : Vendue au-dessous de la valeur nominale, sans paiements périodiques
  • Objectif du REO : Permet de comparer les obligations avec des structures de paiement différentes
  • Base Semestrielle : Le REO suppose une capitalisation semestrielle

Conseils d'Utilisation

  • • Utilisez pour comparer les bons du Trésor avec les obligations de coupon
  • • Un REO plus élevé indique un meilleur rendement pour la comparaison
  • • Considérez les implications fiscales dans les décisions réelles
  • • Le REO ignore le risque de réinvestissement

Comprendre le Rendement Équivalent Obligation

Qu'est-ce que le REO ?

Le Rendement Équivalent Obligation standardise les rendements entre différents types d'obligations en les convertissant sur une base de capitalisation semestrielle, facilitant la comparaison.

Quand Utiliser le REO

  • • Comparer les bons du Trésor avec les obligations
  • • Évaluer différentes options d'investissement
  • • Analyse de rendement de portefeuille

Limitations du REO

  • • Suppose un réinvestissement au même taux
  • • Ignore les différences de risque de crédit
  • • Les considérations fiscales ne sont pas incluses
  • • Les différences de liquidité sont ignorées

Mesures Alternatives

  • • Rendement à l'Échéance (YTM)
  • • Rendement Annuel Effectif
  • • Rendement Équivalent Fiscal
