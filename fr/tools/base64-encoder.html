À Propos de l'Encodage Base64

Encodage : Convertit les données binaires en texte ASCII utilisant 64 caractères (A-Z, a-z, 0-9, +, /)

Rembourrage : Utilise les caractères '=' pour garantir que la longueur de sortie soit un multiple de 4

Augmentation de Taille : L'encodage Base64 augmente la taille d'environ 33 %

Cas d'Utilisation : Pièces jointes e-mail, URL de données, transfert de données API, fichiers de configuration
