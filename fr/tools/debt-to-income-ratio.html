Directives du Ratio d'Endettement

Qualification Hypothécaire :

  • • Prêts conventionnels : ≤ 28% frontal, ≤ 36% dorsal
  • • Prêts FHA : ≤ 31% frontal, ≤ 43% dorsal
  • • Prêts VA : Pas de limite frontale, ≤ 41% dorsal
  • • Prêts Jumbo : ≤ 28% frontal, ≤ 36% dorsal

Directives Générales :

  • • Excellent : 0-20% DTI
  • • Bon : 21-36% DTI
  • • Moyen : 37-43% DTI
  • • Faible : 44%+ DTI
