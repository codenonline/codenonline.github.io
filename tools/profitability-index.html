Formula & Education

Profitability Index Formula:

PI = PV of Future Cash Flows ÷ Initial Investment

Present Value Formula:

PV = CF₁/(1+r)¹ + CF₂/(1+r)² + ... + CFₙ/(1+r)ⁿ

Decision Rules:

  • • PI > 1.0: Accept the project
  • • PI = 1.0: Indifferent (breakeven)
  • • PI < 1.0: Reject the project
  • • Higher PI indicates better investment

Advantages:

  • • Easy to understand and calculate
  • • Considers time value of money
  • • Useful for ranking projects
  • • Relative measure of profitability
