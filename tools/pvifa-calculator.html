PVIFA Formula

PVIFA = [1 - (1 + r)^(-n)] / r

Where:

  • • r = Interest rate per period
  • • n = Number of periods
  • • PVIFA = Present Value Interest Factor of Annuity

What is PVIFA?

PVIFA (Present Value Interest Factor of Annuity) is a factor used to calculate the present value of an ordinary annuity.

Key Points:

  • • Used in time value of money calculations
  • • Assumes payments occur at the end of each period
  • • Higher interest rates result in lower PVIFA values
  • • Longer time periods increase PVIFA values
