How Holding Period Return Works

Basic HPR Formula

HPR = [(Final Value + Dividends - Initial Value) / Initial Value] × 100

Annualized HPR Formula

Annualized HPR = [(1 + HPR)^(1/years) - 1] × 100

Key Components

  • Capital Appreciation: Change in asset value
  • Income: Dividends, interest, or distributions
  • Time Period: Length of investment holding
  • Annualization: Converting to yearly equivalent
