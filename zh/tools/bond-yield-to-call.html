赎回收益率

如果发行人在首个赎回日期提前赎回，确定可赎回债券的实际收益率。

赎回收益率公式

YTC = (C + (CP - P) / n) / ((CP + P) / 2)
其中:
C = 年度票息
CP = 赎回价格
P = 当前债券价格
n = 赎回年限

这是一个近似公式。实际的赎回收益率需要解出使未来现金流现值等于债券价格的折现率。

关键概念

可赎回债券

发行人可在到期前以指定赎回价格赎回的债券。

赎回价格

发行人可赎回债券的价格，通常高于面值。

赎回保护

债券无法赎回的期间，保护投资者免受提前赎回。

赎回风险

当利率下降时债券被赎回的风险，迫使以较低利率进行再投资。

使用提示

  • 评估以溢价交易的可赎回债券时使用赎回收益率
  • 比较赎回收益率与到期收益率以评估赎回可能性
  • 投资时考虑赎回保护期
  • 对可赎回债券考虑再投资风险
  • 审查债券招募说明书中的赎回计划和条款

计算示例

公司债券示例：

  • • 当前价格：$1,080
  • • 赎回价格：$1,050
  • • 年度票息：$60
  • • 赎回年限：3

赎回收益率计算:
YTC = ($60 + ($1,050 - $1,080) / 3) / (($1,050 + $1,080) / 2)
YTC = ($60 - $10) / $1,065 = 4.69%
