解释

破产概率：

Altman Z-Score公式

Z = 1.2A + 1.4B + 3.3C + 0.6D + 1.0E
A： 营运资本 / 总资产
B： 留存收益 / 总资产
C： 息税前收益 / 总资产
D： 权益市场价值 / 总负债
E： 销售 / 总资产

风险区间

安全区（Z > 2.99）
破产风险低。公司财务状况良好。
灰色区（1.8 < Z < 2.99）
中等风险。需要仔细监测。
困境区（Z < 1.8）
高破产风险。需要立即关注。
注意： Altman Z-Score最初是为制造业公司开发的。 针对服务公司和非上市公司的改进版本已存在。

使用提示

数据要求

  • • 使用最近的年度财务报表
  • • 确保所有输入的数据一致性
  • • 营运资本 = 流动资产 - 流动负债
  • • EBIT不包括利息和税费

局限性

  • • 最初为制造业公司设计
  • • 可能不适用于服务公司
  • • 应与其他财务指标一起使用
  • • 市场条件可能影响准确性
