    사용 시기

    • • 같은 산업의 회사들 비교
    • • 적자이지만 성장하는 회사 평가
    • • 매출 기반 평가 분석
    • • 잠재적으로 저평가된 주식 식별
    • • 투자 기회 선별

    장점

    • • 손실을 내는 회사에 적용 가능
    • • 수익 기반 비율보다 변동성 낮음
    • • 자본 구조 비교에 적합
    • • 경기 순환 사업에 유용

    제한 사항

    • • 수익성 차이 무시
    • • 비용 구조 미반영
    • • 성장 전망을 반영하지 않을 수 있음
    • • 자산 집약적 산업에서 오도할 수 있음

    전문가 팁

    • • 의미 있는 분석을 위해 같은 산업 내에서 EV/매출 비율 비교
    • • 성장률 고려 - 빠르게 성장하는 회사의 경우 높은 비율이 정당할 수 있음
    • • 더 현재적인 평가를 위해 지난 12개월(TTM) 매출 사용
    • • EV/EBITDA 및 P/E 비율 등 다른 지표와 결합하여 완전한 분석 수행
    • • 매우 높거나 낮은 비율에 주의 - 근본적인 원인 조사
