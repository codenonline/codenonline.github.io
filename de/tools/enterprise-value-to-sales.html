Wichtigste Erkenntnisse

    EV/Umsatz-Verhältnis verstehen

    Formel

    EV/Umsatz = Unternehmenswert ÷ Jahresumsatz

    EV = Marktkapitalisierung + Gesamtverschuldung - Kasse

    Wann zu verwenden

    • • Vergleich von Unternehmen in der gleichen Branche
    • • Bewertung von unrentablen, aber wachsenden Unternehmen
    • • Analyse von umsatzgestützten Bewertungen
    • • Identifizierung möglicherweise unterbewerteter Aktien
    • • Überprüfung von Investitionsmöglichkeiten

    Vorteile

    • • Funktioniert für verlustmachende Unternehmen
    • • Weniger volatil als gewinnbasierte Kennzahlen
    • • Gut zum Vergleich von Kapitalstrukturen
    • • Nützlich für zyklische Unternehmen

    Einschränkungen

    • • Ignoriert Rentabilitätsunterschiede
    • • Berücksichtigt nicht die Kostenstrukturen
    • • Spiegelt möglicherweise nicht die Wachstumsaussichten wider
    • • Kann für kapitalintensive Branchen irreführend sein

    Profitipps

    • • Vergleichen Sie EV/Umsatz-Verhältnisse innerhalb der gleichen Branche für aussagekräftige Analysen
    • • Berücksichtigen Sie Wachstumsraten - höhere Verhältnisse können für schneller wachsende Unternehmen gerechtfertigt sein
    • • Verwenden Sie den Umsatz der letzten zwölf Monate (TTM) für aktuellere Bewertungen
    • • Kombinieren Sie mit anderen Kennzahlen wie EV/EBITDA und KGV für eine vollständige Analyse
    • • Vorsicht bei extrem hohen oder niedrigen Verhältnissen - untersuchen Sie die zugrunde liegenden Gründe
