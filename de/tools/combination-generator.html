Frequently asked questions
What is the difference between combinations and permutations?
Combinations are selections where order doesn't matter (e.g., choosing 3 team members). Permutations are arrangements where order matters (e.g., race positions where 1st, 2nd, 3rd are different).
How many combinations of 3 items can you make from 5?
Using the formula C(5,3) = 10 unique combinations. This is calculated as 5! / (3! × 2!) = 120 / (6 × 2) = 10.
What does allow duplicates mean in combinations?
Allow duplicates (combinations with repetition) means the same item can appear multiple times in a combination. Without this option, each item can only be selected once.
What are real-world uses for combination generators?
Common uses include team assembly from available members, quality testing with different parameter combinations, creating menu variations, and generating product bundle options for e-commerce.
Can the number of combinations exceed the number of items?
Yes, absolutely. For example, 5 items with combination size 3 produces 10 combinations. With repetition allowed, the numbers grow even larger as items can repeat.
