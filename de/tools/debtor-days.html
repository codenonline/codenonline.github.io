Formel

Debitorentage = Durchschnittliche Forderungen
Tägliche Kreditverkäufe
Tägliche Kreditverkäufe = Jährliche Kreditverkäufe ÷ Zeitraum (Tage)

Dabei gilt:

  • Durchschnittliche Forderungen: Durchschnittlicher Betrag, den Kunden schulden
  • Jährliche Kreditverkäufe: Gesamte Kreditverkäufe für den Zeitraum
  • Zeitraum: Anzahl der Tage im Zeitraum (typischerweise 365)

Bedienungsanleitung

Was wird gemessen:

Debitorentage (auch als Tage des Verkaufsausgleichs bezeichnet) messen die durchschnittliche Anzahl von Tagen, die für den Einzug von Forderungen von Kunden erforderlich sind.

So berechnen Sie:

  1. Geben Sie Ihren durchschnittlichen Forderungssaldo ein
  2. Geben Sie Ihre jährliche Kreditverkaufszahl ein
  3. Wählen Sie den geeigneten Zeitraum
  4. Überprüfen Sie die berechneten Debitorentage

Interpretation:

  • Niedriger ist besser: Schnellerer Einzug von Forderungen
  • Mit Zahlungsbedingungen vergleichen: Sollte mit Zahlungsbedingungen übereinstimmen
  • Branchenvergleich: Vergleich mit Wettbewerbern
  • Trendanalyse: Überwachen Sie Änderungen im Zeitverlauf

Beispielrechnung

Beispielunternehmen:

Durchschnittliche Forderungen: $50,000
Jährliche Kreditverkäufe: $365,000
Zeitraum: 365 Tage
Tägliche Kreditverkäufe = 365.000 EUR ÷ 365 = 1.000 EUR
Debitorentage = 50.000 EUR ÷ 1.000 EUR = 50 Tage

Das bedeutet, dass Kunden durchschnittlich 50 Tage benötigen, um ihre Rechnungen zu bezahlen, was als durchschnittliche Leistung gilt.

Verbesserungstipps

Debitorentage reduzieren:

  • Bieten Sie Skonto für Frühzahlung an (2/10 netto 30)
  • Implementieren Sie strengere Kreditrichtlinien
  • Senden Sie Rechnungen sofort nach Lieferung
  • Folgen Sie überfälligen Konten zeitnah nach

Einzugsstrategien:

  • Automatisieren Sie die Rechnungsverarbeitung und Erinnerungen
  • Verlangen Sie Anzahlungen für große Bestellungen
  • Nutzen Sie Factoring für sofortigen Cashflow
  • Überprüfen Sie die Kreditwürdigkeit von Kunden regelmäßig

Überwachung:

  • Überwachen Sie monatlich, um Trends zu erkennen
  • Segmentieren Sie nach Kundentyp oder Region
  • Vergleichen Sie mit Branchen-Benchmarks
  • Legen Sie Verbesserungsziele fest
